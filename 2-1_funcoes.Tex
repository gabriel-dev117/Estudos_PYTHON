Em Python, eu consigo alterar o valor de uma variável
externa ao escopo, utilizando "global" e a variável
Ex.:

|    x = 1
| 
|    def escopo():
|        global x
|        x =10
|
|        def outra_funcao():
|            y = 2
|            print(x, y)
|        outra_funcao()
|        print(x)
|
|
|    print(x)
|    escopo()
|    print(x)

Neste caso, o "x" será alterado dentro da função escopo
por causa do "global"
